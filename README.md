# 🎂 李星娅生日祝福网站 - 星空寄语

> 为李星娅量身定制的3D星空生日祝福网站

## ✨ 功能特色

- 🌟 **3D星空场景** - 基于Three.js的交互式星空
- 🦁 **狮子座高亮** - 李星娅的生日星座特别展示  
- 📸 **个人回忆** - 9颗星星对应的照片和回忆
- 🎵 **背景音乐** - 《Theme of Exodus》伴奏
- 📱 **移动适配** - 完美支持手机和平板
- 🎭 **优雅动画** - 流畅的光晕和交互效果

## 🚀 快速开始

### 在线访问
访问 GitHub Pages 部署地址

### 本地运行
```bash
# 使用Node.js服务器
npm run dev

# 或使用Python服务器  
python -m http.server 8000
```

## 🎮 使用方法

1. 🖱️ **鼠标/触摸交互** - 拖动旋转星空，缩放查看
2. ⭐ **点击星星** - 第一次点击显示生日祝福，后续点击查看回忆
3. 🎵 **音乐控制** - 点击右上角按钮控制背景音乐
4. 📊 **进度显示** - 左上角显示已查看的星星数量
